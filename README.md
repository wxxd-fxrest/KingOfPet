## 멍냥상전 - King of Pet
React-Native + Firebase + Api (05)

> 주요기능

+ 다른 사용자들과의 소통
+ QnA, 건강 및 기타 궁금증에 대한 질문과 답변
+ Diary, 하루 일과 또는 특별한 일에 대한 기록
+ 팔로우, 언팔로우
+ 사용자, 게시글 검색
+ 회원가입 / 로그인
  소셜 로그인
  회원 정보 수정
